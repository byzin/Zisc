# Zisc #

Zisc provides commonly used C++ classes and functions.

Zin's miscellaneous C++ library

Version: 0.0.50

## Requirement ##

### Compiler ###

* C++17 support compiler

### Build tools ###

* [CMake](https://cmake.org/) (version 3.12 or later)
